# Change Log

All notable changes to the "Placeholder SVG (Frontmatter)" extension will be documented in this file.

## [0.0.1] - 2026-02-04

### Added
- Initial release
- Placeholder SVG: Insert with custom dimensions (width×height)
- Placeholder SVG: Insert with aspect ratio (1:1, 4:3, 4:5, 16:9)
- Quick presets: Hero (1440×720), Card (400×300), Avatar (128×128)
- Landscape/Portrait orientation support
- Width-first and height-first dimension calculation
- Memory-efficient SVG caching
- Remembers last used dimensions and preferences
- Context menu integration in editor

### Features
- Generates deterministic SVG placeholder images as data-URIs
- Supports up to 8192×8192 dimensions
- Real-time validation of inputs
- User-friendly error messages
- Cross-platform compatibility (Windows, macOS, Linux)
